### `recommendation_engine.py`

#### 함수: `_create_reason()`

**주요 변경:**

1. **거리 정보 제거**
```python
# 제거됨
# distance = place.get('distance', 0)
# if distance < 5:
#     reasons.append(f"기준 위치에서 {distance:.1f}km로 매우 가까워요")
```

2. **키워드 추출 및 표시**
```python
keywords = place.get('keywords', '')
if keywords:
    keyword_list = [k.strip() for k in str(keywords).split(',')[:3]]
    keyword_text = ', '.join(keyword_list)
    
    if keyword_similarity > 0.6:
        reasons.append(f"선호하시는 {keyword_text} 스타일과 매우 잘 맞는 곳입니다 ✨")
```

3. **자연스러운 문장**
```python
# 변경 전
"아침 커피로 시작하기 좋은 곳이에요"

# 변경 후
"아침 시작을 상쾌하게 해줄 커피와 분위기가 좋은 곳입니다"
```

4. **AI 추천 처리**
```python
is_ai = place.get('is_ai_recommended', False)
if is_ai:
    return f"🤖 AI 분석 결과: {reasons[0]}"
```
