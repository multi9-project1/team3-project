# 🍊 제주 여행 맞춤 추천 시스템 v4.1

> **팀 3 프로젝트** | 피수민 · 안성민 · 오경석 · 김구  
> CSV 데이터 기반 장소 추천 + 카카오 API 실시간 경로 + OpenAI 챗봇

---

## 📌 프로젝트 개요

제주도 여행자를 위한 **맞춤형 코스 추천 웹 앱**입니다.  
팀이 직접 수집한 약 **1,269개 제주 장소 데이터(CSV)** 를 기반으로 평점·리뷰·카테고리·키워드를 분석해 최적 코스를 자동 생성하며, 카카오 API로 실시간 경로 정보를, OpenAI API로 AI 챗봇 기능을 제공합니다.

## 🗂️ 파일 구조

```
jeju_v4/
├── app.py                   # 메인 Streamlit 앱 (UI · 세션 관리 · 탭 렌더링)
├── config.py                # 전역 설정 (API 키 로드 · 카테고리 · 시간대 슬롯)
├── data_manager.py          # CSV 로딩 · 필터링 · 통계
├── recommendation_engine.py # 추천 로직 (자동/직접 구성 · 점수 계산)
├── kakao_service.py         # 카카오 REST API 래퍼 (숙소 검색 · 경로 · 거리)
├── ui_components.py         # 일차별 탭 · 전체 지도 · 코스 분석 렌더링
├── chatbot.py               # OpenAI 기반 AI 여행 챗봇
├── data.csv                 # 제주 장소 데이터 (~1,269개)
├── requirements.txt         # Python 패키지 목록
└── .env.example             # API 키 환경변수 샘플
```

## 🧮 추천 점수 계산 방식

```
점수 = 평점 × 10
     + min(리뷰수 / 10, 20)       # 리뷰 가중 (최대 20점)
     + 슬롯 키워드 매칭 × 5점
     + 사용자 선호 키워드 매칭 × 10점 (리뷰 텍스트 매칭 시 +3점)
     - 거리(km) × 0.5점           # 거리 패널티
```

## ⚙️ 설치 및 실행

### 1. 패키지 설치

```bash
pip install -r requirements.txt
```

### 2. API 키 설정

`.env.example` 을 복사해 `.env` 파일을 생성한 뒤 API 키를 입력합니다.

```bash
cp .env.example .env
```

```env
KAKAO_API_KEY=여기에_카카오_REST_API_키_입력
OPENAI_API_KEY=여기에_OpenAI_API_키_입력   # 선택 — 없어도 기본 추천 동작
OPENAI_MODEL=gpt-4o-mini
```

> **또는** `.env` 없이 앱 실행 후 사이드바에서 직접 입력해도 됩니다.

### 3. 앱 실행

```bash
streamlit run app.py
```
